## Commands

```rust
// Check version
rustc --version

// Update to latest version
rustup update

// Open local Rust docs
rustup doc

// Compile a Rust program
rustc main.rs
// => ./main

// New Cargo project
cargo new hello_cargo

// Compile with Cargo
cargo build

```
